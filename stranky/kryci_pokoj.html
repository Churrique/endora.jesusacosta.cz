¿Qué es Markdown?...

Markdown es un lenguaje de marcado ligero que puede usar para agregar elementos de formato a documentos de texto sin formato. Markdown, creado por John Gruber en 2004, es ahora uno de los lenguajes de marcado más populares del mundo. Un ejemplo de su uso está en los repositorios de Git Hub. Si quieres practicar un poco, aquí tienes un editor en línea.

¿Quieres aprender HTML?...

HTML significa, del inglés, HyperText Markup Language (Lenguaje de Marcas de Hipertexto). Es el componente más básico de la Web. Define el significado y la estructura del contenido web. "Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí, ya sea dentro de un único sitio web o entre sitios web. Existen un sin fin de editores en línea, en los que te puedo ofrecer: LiveWeave ó JS Bin ó JSFiddle ó Code Pen.

¿Quieres aprender CSS?.

CSS es acrónimo de "Cascading Style Sheets", lo que significa hojas de estilo en cascada. Tanto "HTML" como "CSS", son dos de las tecnologías centrales para crear páginas web. "HTML" proporciona la estructura de la página, y "CSS" el diseño tanto visual como también auditivo, para una gran variedad de dispositivos.

Por supuesto que te puede ser útil.
